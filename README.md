# Basic LAN Network using Cisco Packet Tracer

## Overview
This project demonstrates the setup of a basic Local Area Network (LAN) using Cisco Packet Tracer.

## Network Design
- 3 PCs connected to a switch
- All devices are in the same network

## Configuration
- PC0: 192.168.1.1
- PC1: 192.168.1.2
- PC2: 192.168.1.3
- Subnet Mask: 255.255.255.0

## What I did
- Added end devices (PCs) and a switch
- Connected devices using copper straight-through cables
- Assigned static IP addresses manually
- Tested connectivity using ping

## Result
All devices successfully communicated with each other using ICMP (ping).
